## How to Add Images

### Option 1: Use AI Image Generators
- Use tools like Midjourney, DALL-E, or Stable Diffusion
- Use the prompts above
- Export as high-quality images
- Convert to WebP format
- Save to `assets/images/` folder

### Option 2: Use Stock Photos
- Search on Unsplash, Pexels, or similar
- Look for SaaS dashboard, tech product mockups
- Edit/composite as needed
- Convert to WebP
- Save to `assets/images/` folder

### Option 3: Create Custom Mockups
- Use Figma/Adobe XD to design interfaces
- Export as PNG
- Convert to WebP using online tools or:
  ```bash
  # Using cwebp (install via npm or system package manager)
  cwebp input.png -o output.webp -q 80
  ```
